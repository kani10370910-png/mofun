# 活动设计 PRD 文档

> 模块路径：品牌设计 → 活动　|　页面 `/image?sub=event`

---

## 功能概述

**活动设计**：通过「文生图 / 图生图」两种方式，结合**成图类型**（海报 / 长图 / 菜单 / 易拉宝 / 宣传单）、**图片尺寸**、**画面风格**，把简短的活动主题扩写并出图，产出可二次编辑的活动物料。右侧「生成历史 / 参考灵感」双 Tab，参考灵感内置真实样张可一键套用。

**模块说明**

- 左侧表单分「文生图」「图生图」两个 Tab，顶部「文生图 / 图生图」切换 + 「成图类型」固定吸顶。
- 文生图：填画面描述 → 出图前按成图类型专属系统提示词扩写 → 调文生图模型出图。
- 图生图：上传 / 仓库选参考图 + 修改需求（可选图片处理预设）→ 据参考图改图。
- 右侧「生成历史」展示进度卡与结果图（支持只看收藏 / 编辑 / 深度编辑 / 下载 / 收藏 / 另存）；「参考灵感」展示真实样张，可一键套用。

## 1. 文生图（文生图 Tab）

### 1.1 业务字段

| 字段 | 必填 | 类型 | 说明 |
|---|---|---|---|
| 成图类型 | 是 | 单选 chip | 自定义 / 海报 / 长图 / 菜单 / 易拉宝 / 宣传单；切换时联动「图片尺寸」重置为该类型尺寸组首项 |
| 画面描述 | 是 | 多行文本 | 活动主题、主体、风格、氛围、文案；支持「联想」「图转文」「清空」工具按钮 |
| 图片尺寸 | 是 | 下拉 | 按成图类型给不同尺寸组；选「自定义」展开宽 × 高（px）输入框 |
| 生图模型 | 是 | 下拉 | MoFun 区域文化大模型等 |
| 生成数量 | 是 | 分段 | 1 / 2 / 4，默认 4 |
| 画面风格 | 是 | 卡片→九宫格 | 智能匹配 / 国潮 / 水彩 / 扁平矢量 / 卡通动漫 / 线描 / 版画 / 写实摄影 / 纹样 |

**画面描述工具栏**

- **联想**：调用 LLM 把当前描述扩写得更丰富（约 300 字，前端按句末标点收敛，不切半句），流式回填输入框。
- **图转文**：右侧弹出面板，上传图片由视觉模型反推出画面描述词，回填输入框。
- **清空**：清空画面描述。

> 说明：联想 / 图转文得到的描述已是「扩写成品」，立即生成时不再二次扩写（见 1.4）。

### 1.2 成图类型 → 尺寸组对应

| 成图类型 | 尺寸组 | 典型尺寸 |
|---|---|---|
| 海报 | posterRatios | 自定义 / 竖版9:16 / 横版16:9 / 竖版3:4 / 方版1:1 / 电商海报 / 标准长图 |
| 易拉宝 | rollupRatios | 自定义 / 标准易拉宝(2m·1.8m) / X展架 / 门型展架 |
| 宣传单 | flyerRatios | 自定义 / A4单页 / A5单页 |
| 长图 / 菜单 / 自定义 | imageRatios（通用） | 自定义 / 正方形1:1 / 纵向3:5 / 横向5:3 / 宽屏16:9 |

> 尺寸下拉首项均含「自定义」；「自定义」时由宽 × 高输入框决定出图尺寸。文生图模型要求图 ≥ 369 万像素，出图时自动按比例放大到该下限。

### 1.3 交互流程

1. 选成图类型 → 图片尺寸联动重置。
2. 填画面描述（或点联想扩写 / 图转文反推）。
3. 选图片尺寸 / 生图模型 / 生成数量 / 画面风格。
4. 点「立即生成」→ 切到「生成历史」→ 出进度卡 →（非套用场景）按成图类型系统提示词扩写描述 → 调文生图模型出图 → 进度卡替换为结果图。
5. 出图后自动存入「仓库 · 我的作品」。

### 1.4 设计提示词（按成图类型扩写）

立即生成时，文生图先用「成图类型」专属系统提示词把简短描述扩写成完整画面描述词，再送文生图模型。占位符按当前表单填充。

```
【输入参数】
用户描述：{{ user_input }}
县域/地区：{{ county_name }}（若为空则不强制融入地域元素）
图片比例：{{ image_ratio }}
画面风格：{{ art_style }}

【扩写规则（以海报为例）】
1. 输出为纯中文，不含英文。
2. 按【图片比例】匹配构图：
   1:1 → 居中构图，四边留白均匀
   3:4/4:5 → 上方三分之二主视觉，下方文字信息区
   9:16 → 顶部品牌区 / 中部主体 / 底部行动召唤区
   16:9 → 左右分区横向构图，右侧留文字排版空间
3. 按【画面风格】注入风格关键词（国潮 / 水彩 / 扁平矢量 / 卡通动漫 / 线描 / 版画 / 写实摄影 / 纹样 / 智能匹配）。
4. 内容顺序：画面主体 → 场景环境与光影 → 风格关键词 → 色调情绪 → 构图描述 → 文字排版提示 → 画面质量，用逗号连接为一段。
5. 县域非空时融入地域景观 / 农特产 / 民俗。
6. 保留用户输入中的主标题原文，注明文字内容与排版位置。
7. 只输出最终画面描述段落，不加解释。
8. 海报 150~250 字；长图 / 菜单 / 易拉宝 / 宣传单 180~280 字。
```

> 长图 / 菜单 / 易拉宝 / 宣传单 各有同结构、按物料特性定制的系统提示词（分区节奏、菜品陈列、三区结构、信息密度等）。
>
> **跳过扩写的场景（fromCase）**：套用参考灵感（样张提示词已完整）、图转文回填、联想回填——这三类描述本身就是成品，立即生成时直接出图，不再二次扩写；一旦用户手动改描述或切成图类型，恢复扩写。

### 1.5 联想扩写收敛逻辑

- 系统提示词要求约 300 字（模型常超量，500+ 字）。
- 前端落定时 `clampAssoc`：在约 300 字附近找最近的句末标点（。！？…；）收尾，找不到退而求逗号，绝不切在半个字 / 半句；最终约 270~330 字。

## 2. 图生图（图生图 Tab）

### 2.1 业务字段

| 字段 | 必填 | 类型 | 说明 |
|---|---|---|---|
| 参考图片 | 是 | 上传 / 仓库 | 本地上传（jpg/png）或点「仓库」从「我的素材 / 作品」选图；带预览与清除 |
| 修改需求 | 是 | 多行文本 | 描述对参考图的改动；可由「图片处理预设」一键填入 |
| 图片处理预设 | 否 | 网格单选 | 生成相似图 / 细节修复 / 变清晰 / 去文字 / 去水印 / 换风格 / 相似字体 / 删除元素 / 增加元素 / 改尺寸 / 提取线稿（更多预设可展开） |
| 参考强度 | 条件 | 滑块 | 仅「生成相似图」预设时出现：弱 / 中 / 强 |
| 编辑模型 | 是 | 下拉 | 基础编辑模型 / 高清重绘模型 / 局部重绘模型 |

### 2.2 图片处理预设 → 提示词

| 预设 | 填入「修改需求」 |
|---|---|
| 生成相似图 | 生成相似图 |
| 细节修复 | 细节修复 |
| 变清晰 | 将这张图片变清晰 |
| 去文字 | 去除图片中的所有文字 |
| 去水印 | 去除画面中的水印 |
| 换风格 | 将图片改为XX风格 |
| 相似字体 | 按照图片中的字体生成"XXXX"文字 |
| 删除元素 | 删除XX内容 |
| 增加元素 | 在图片的XX位置增加一个XX内容 |
| 改尺寸 | 将图片扩展为XX:XX的尺寸 |
| 提取线稿 | 将图片转为黑白线稿图 |

> 「不使用预设」或再次点击已选项 → 取消选中并清空修改需求。

### 2.3 交互流程

1. 上传 / 仓库选参考图（必填）。
2. 选图片处理预设（自动填入修改需求）或手填修改需求。
3.（生成相似图）调参考强度。
4. 点「立即生成」→ 参考图转 data URL 作为 `image` 参数，连同修改需求送文生图模型据图改图（一次出 1 张）。

## 3. 右侧画廊：生成历史 / 参考灵感

### 3.1 生成历史

| 元素 | 说明 |
|---|---|
| 只看收藏 | 开关，仅显示已收藏的结果图 |
| 分组 | 按生成时间分「今天 / 昨天 / 更早」 |
| 行头 | 画面描述 + 成图类型标签 + 复制 / 删除 + 生成时间 |
| 进度卡 | 出图中显示「X% 完成 · 正在构思」 |
| 结果图 | 按宽铺满，hover 从上滚到下展示全图；hover 出「编辑 / 深度编辑」按钮 |
| 结果图操作 | 收藏（右上心形）、下载、另存为我的素材（右下） |

- 结果图点击（非按钮处）放大预览。
- 出图自动存入「仓库 · 我的作品」。

### 3.2 参考灵感

| 元素 | 说明 |
|---|---|
| 样张卡片 | 真实样张图，按宽铺满，hover 从上滚到下展示全图；左上角成图类型标签 |
| 套用模版 | hover 出现，点击：回填画面描述 + 成图类型 + 原图尺寸（自定义宽高），标记 fromCase 跳过扩写 |
| 点击卡片 | 左侧成图类型 + 原图尺寸跳转（不回填描述） |

- 样张数据含成图类型（sub）、提示词（prompt）、原图宽高（w / h）；套用时用原图宽高设为「自定义」尺寸，100% 还原比例。

## 4. 图片编辑（结果图「编辑」入口）

点结果图「编辑」进入编辑工作台：

| 区域 | 内容 |
|---|---|
| 顶栏 | 返回 / 重置原图 / 对比原图（按住）/ 深度编辑 |
| 左工具列 | AI变清晰 / AI消除 / AI抠图 / AI扩图 / 细节修复 / 转矢量 |
| 中画布 | 图片按比例完整显示（flex 居中）；AI扩图时显示贴合图片的裁剪框 |
| 底部操作条 | 随工具变化（放大倍数 2X/4X、画笔粗细、抠图模型、扩图宽高等）|
| 右历史记录 | 一行 2 个记录卡，带 SVG/PNG 角标 + 另存为素材 / 下载 |

---

## 5. 深度编辑（结果图「深度编辑」入口）

把一张海报转为分层可编辑的平面画布。

**流程动画**：解析海报 → 海报图文分层 → 刀版图生成中 → 进入画布。

**平面画布**

| 区域 | 内容 |
|---|---|
| 顶栏 | 文字 / 图片 / 素材 / 生成 四个下拉面板 + 下载设计文件 / 保存 |
| 左图层 | 可折叠/回拉展开的图层列表；每条 hover 出三点菜单（复制 / 删除 / 移到最前 / 移到最后 / 重命名 / 隐藏 / 锁定）+ 眼睛显隐 |
| 中画布 | 海报分层预览 + 缩放条 |
| 右属性 | 画布/图片属性（尺寸 / 填充 / 蒙版 / 旋转）+ 图片编辑工具网格 |

## 附：依赖接口

| 接口 | 用途 | 说明 |
|---|---|---|
| `/api/image` | 文生图 / 图生图出图 | doubao-seedream 系列；图生图带 `image`（data URL）参数；尺寸 ≥ 369 万像素 |
| `/api/generate` | 联想扩写 / 文生图扩写 | SSE 流式；scene = `t2i-associate` / `t2i-event` |
| `/api/vision` | 图转文 | 视觉模型看图反推画面描述 |
| `/api/proxy-image` `/api/download` | 结果图下载 | 跨域代理拉取 |

> 部署注意：静态导出（nginx 静态站）不含 `/api`，文生图 / 联想 / 图转文等接口功能不可用；需出图的环境须用 Node 全栈部署。
